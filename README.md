# SkillSwitch

> Visual skill manager for AI agents. Toggle skills on/off with a switch.

## Features

- 📦 See all installed skills at a glance
- ⚡ Toggle skills on/off with one click
- 🎯 Keep your AI workspace focused and clutter-free
- 🚀 No restart needed

## Current Support

- **Claude Code** - `~/.claude/skills/`

## Planned

- Cursor
- GitHub Copilot
- More AI agents

## Development

```bash
# Install dependencies
npm install

# Run in development mode
npm run electron:dev

# Build for production
npm run electron:build
```

## Tech Stack

- **Framework**: Electron 31
- **Frontend**: React 18 + TypeScript + Vite 5
- **Packaging**: electron-builder
